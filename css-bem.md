### BEM
- block, element, modifier
- Css class를 적용하는 규칙
- 프로젝트의 css를 체계적으로 작성할 수 있게 해주며, 이를 통해 성능을 높일 수 있고, 유지 보수성을 높일 수 있도록 도와준다.
